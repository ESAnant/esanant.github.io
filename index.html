FPGA Hardware Accelerator for MLP Neural Network

Developed a comprehensive hardware accelerator on the Xilinx Zynq-7000 FPGA to improve MLP neural network inference. Implemented designs in software, HLS, and Verilog, focusing on prediction accuracy. Executed profiling and optimization utilizing pipelining, loop unrolling, and array partitioning, achieving significant speed and efficiency gains.

FPGA Xilinx Zynq-7000 Verilog HDL HLS Neural Networks Hardware Acceleration

In-Memory Compute Circuit for Neural Network Acceleration

Designed an advanced in-memory compute circuit using NeuroSim and MuMax3 to accelerate neural network computations. Concentrated on quantization and optimization techniques to boost accuracy and efficiency. Conducted comprehensive simulations in PyTorch, comparing different data types (float64, float32, int8, int4) to analyze model performance and resource utilization.

NeuroSim MuMax3 PyTorch In-Memory Computing Quantization Neural Networks

VLSI Interconnect Modelling and Simulation

Engaged in optimizing processor interconnects using Elmore RC models and Cadence Virtuoso for a 2-core processor at 45nm technology node. Focused on energy-delay tradeoffs to improve system efficiency, combining theoretical knowledge with practical implementation to address VLSI design challenges and improve signal integrity and system performance.

Cadence Virtuoso VLSI Design 45nm Technology Signal Integrity RC Modeling

Digital Circuit Design and Standard Cell IP Development

Contributed to the creation of a ring oscillator Standard Cell IP leveraging hierarchical design in Cadence Virtuoso, targeting efficiency in 40nm technology with focus on minimizing area while optimizing for PVT variations. Ensured design integrity through strict adherence to DRC and LVS standards.

Standard Cell Ring Oscillator 40nm Technology PVT Analysis DRC/LVS
